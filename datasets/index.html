LineEX

公開リポジトリの sample_input と sample_output から少数サンプルを保存。

アノテーション / 出力として分かるもの

  • 線系列ごとのデータ点列
  • 検出・対応付け後の可視化画像
  • 各点は画像座標 (x, y)
pred_data_0.png.json { "0.png": [ [[99, 580], [99, 594], [221, 562], ...], [[763, 228]], [], [[99, 359], [141, 354], [178, 353], ...] ] }

上のように、1 枚の画像に対して系列ごとの点列が入っています。

ExcelChart400K

公式公開先は巨大な配布物（Hugging Face / Drive 上の multi-GB zip）。今回は関連実装 DeepRule / ChartOCR リポジトリに含まれるデモ画像を保存。

公開情報から分かる代表的な注釈

  • Line: 折れ線のデータ点列 [x1, y1, x2, y2, ...]
  • Bar: 棒の bounding box
  • Pie: 扇形を定義する 3 点 [center_x, center_y, edge1_x, edge1_y, edge2_x, edge2_y]
  • Cls / components: タイトル、凡例、描画領域などの bbox

今回は本体の巨大 zip までは展開せず、まず関連デモ画像を保存しています。

UB-PMC / CHART-Info

CHART-Info 公式 sample zip UB_PMC_sample_1.21.zip をダウンロードして展開。

サンプル JSON に含まれる注釈

  • task1: Chart Classification
  • task2: Text Detection and Recognition（OCR テキスト + polygon）
  • task3: Text Role Classification（tick label / legend label など）
  • task4: Axes Analysis（軸、tick point、plot 領域）
  • task5: Legend Analysis（凡例記号 bbox とラベル対応）
  • task6: Data Extraction（系列名、値、visual elements）

つまり、チャート種別 → OCR → テキスト役割 → 軸 → 凡例 → 最終データ抽出まで段階的に注釈されています。
