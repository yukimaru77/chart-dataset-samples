# Model example artifacts

This directory contains example inputs / outputs copied out of the local model workbench so they are visible from the parent repository.

- `ChartOCR/`: repo-provided demo images
- `Kato2022/`: locally generated chart synthesizer output
- `LineEX/`: repo-provided sample input/output/json
- `LineFormer/`: repo-provided demo input/output
- `DePlot/`: local DePlot demo input and output text
- `OneChart/`: repo-provided visual examples
- `ChartVSR/`: no public runnable artifact captured yet
